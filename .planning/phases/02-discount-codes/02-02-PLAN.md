```typescript
export const expensesApi = createApi({
  reducerPath: 'expensesApi',
  baseQuery: fetchBaseQuery({ baseUrl: '/api' }),
  tagTypes: ['Expenses'],
  endpoints: (builder) => ({
    getExpenses: builder.query<Expense[], void>({ query: () => 'expenses', providesTags: ['Expenses'] }),
    addExpense: builder.mutation<Expense, Partial<Expense>>({ query: (data) => ({ url: 'expenses', method: 'POST', body: data }), invalidatesTags: ['Expenses'] }),
    updateExpense: builder.mutation<Expense, Partial<Expense> & { id: string }>({ ... }),
    deleteExpense: builder.mutation<void, string>({ query: (id) => ({ url: `expenses/${id}`, method: 'DELETE' }), invalidatesTags: ['Expenses'] }),
  }),
})
```

```sql
CREATE TABLE IF NOT EXISTS bsp_discount_codes (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    code_name TEXT NOT NULL UNIQUE,
    amount NUMERIC(10, 2) NOT NULL,
    is_active BOOLEAN NOT NULL DEFAULT true,
    created_at TIMESTAMPTZ DEFAULT now(),
    updated_at TIMESTAMPTZ DEFAULT now()
);
```

**1. Create src/lib/api/discountCodesApi.ts**

Define a DiscountCode interface with fields matching the DB table:
```typescript
export interface DiscountCode {
  id: string
  code_name: string
  amount: number
  is_active: boolean
  created_at: string
  updated_at: string
}
```

Create `discountCodesApi` using `createApi` with:
- `reducerPath: 'discountCodesApi'`
- `tagTypes: ['DiscountCodes']`
- `getDiscountCodes`: GET 'discount-codes', ordered by created_at desc — `providesTags: ['DiscountCodes']`
- `addDiscountCode`: POST 'discount-codes' with `{ code_name, amount }` body — `invalidatesTags: ['DiscountCodes']`
- `updateDiscountCode`: PATCH `discount-codes/${id}` with `Partial<DiscountCode> & { id: string }` — `invalidatesTags: ['DiscountCodes']`
- `deleteDiscountCode`: DELETE `discount-codes/${id}` — `invalidatesTags: ['DiscountCodes']`

Export all four hooks: `useGetDiscountCodesQuery`, `useAddDiscountCodeMutation`, `useUpdateDiscountCodeMutation`, `useDeleteDiscountCodeMutation`.

**2. Update src/lib/store.ts**

Import `discountCodesApi` from `'./api/discountCodesApi'`. Add `[discountCodesApi.reducerPath]: discountCodesApi.reducer` to reducer object. Add `discountCodesApi.middleware` to the middleware concat chain.

**3. Create src/app/api/discount-codes/route.ts**

GET handler: `supabaseAdmin.from('bsp_discount_codes').select('*').order('created_at', { ascending: false })`. Return array or error.

POST handler: validate `code_name` (required, non-empty string) and `amount` (required, positive number). Insert into bsp_discount_codes. Return 201 with created record.
- `code_name` should be stored as uppercase to ensure case-insensitive matching: `code_name.trim().toUpperCase()`

**4. Create src/app/api/discount-codes/[id]/route.ts**

PATCH handler: accepts `{ code_name?, amount?, is_active? }`. Updates record by id. Also update `updated_at: new Date().toISOString()`. Return updated record.

DELETE handler: deletes record by id. Return `{ success: true }`.

Both handlers use `supabaseAdmin` (import from `'@/lib/supabaseAdmin'`).

Error handling follows existing pattern: wrap in try/catch, return `NextResponse.json({ error: '...' }, { status: NNN })`.
